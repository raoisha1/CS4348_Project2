Project Overview

This project implements several multithreading and synchronization programs.

Files included:
	1.	thread_demo.py – Demonstrates basic thread creation and mutual exclusion with semaphores.
	2.	example.py –  sing semaphores until a maximum value is reached.
	3.	bank.py – A full bank simulation involving multiple tellers and customers, shared resources (manager and safe), and synchronization using semaphores.
	4.	devlog.md – Session-by-session development log following project requirements.
